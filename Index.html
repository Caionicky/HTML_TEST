HTML: A Linguagem da Web

HTML é a linguagem usada para estruturar páginas na internet.

História do HTML

O HTML foi criado por Tim Berners-Lee em 1991. Desde então, passou por várias versões.

"HTML é a espinha dorsal da web moderna."
